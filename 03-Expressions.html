Expression

  • "Expressions" são usadas para vincular os dados do aplicativo com o HTML
  • "Expressions" são representadas por dois pares de chaves "{{ expression}}".
  • O comportamento é o mesmo que a diretiva ng-bind
  • "Expressions" em AngularJS nada mais são que expressões javascript que são processados e seus valores são impressos no html
